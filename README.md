# Lean Six Sigma Calculators

This repository contains practical Lean and Six Sigma calculators designed to support real operational improvement work.

Available tools include:

- Takt Time Calculator
- OEE Calculator
- DPMO Calculator
- Sigma Level Calculator

These tools are used by operations, manufacturing and business improvement teams to measure performance and support delivery.

For the full suite of calculators, templates and training resources, visit:
