# Lean Six Sigma Calculators

This repository contains a practical library of Lean and Six Sigma calculators designed to support real operational improvement work.

These tools are used by operations, quality, manufacturing and business improvement professionals to measure performance, analyse variation and support structured delivery from Define through to Control.

All calculators are built for practical use in day-to-day improvement environments and are intended to simplify complex analysis into clear, actionable results.

---

## Available Calculators

This repository currently includes the following tools:

- Takt Time Calculator  
- Overall Equipment Effectiveness (OEE) Calculator  
- Defects Per Million Opportunities (DPMO) Calculator  
- Sigma Level Calculator  
- Rolled Throughput Yield (RTY) Calculator  
- First Pass Yield (FPY) Calculator  
- Defects Per Unit (DPU) Calculator  
- Process Capability (Cp) Calculator  
- Process Capability (Cpk) Calculator  
- Control Limits Calculator

- Histogram Builder

- Standard Error Calculator  
- Confidence Interval Calculator  
- Z Score Calculator  
- Process Performance (Pp / Ppk) Calculator  
- Cost of Poor Quality (COPQ) Calculator  
- Variation and Stability Analysis Tools

---

## How to Use

Each calculator is provided as a standalone HTML file and can be opened directly in any modern web browser.

No installation is required.

Simply:

1. Open the calculator file  
2. Enter your process data  
3. Review the calculated results  
4. Use the outputs to support improvement decisions
